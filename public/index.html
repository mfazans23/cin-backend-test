API Documentation

This API provides endpoints to register a user, login, and access user information.

Stack Used

  • Express.js
  • TypeScript
  • Sequelize (with MySQL)
  • Bcrypt (for password encryption)
  • JSON Web Tokens (JWT) for authentication

Endpoints

The API has the following endpoints:

POST /api/register

Register a new user with the API. Use this endpoint to create a new user account.

Request Body: 

{
  "name": "John Doe",
  "email": "johndoe@example.com",
  "password": "password123"
}

Response (Success): 

{
  "message": "User registered successfully"
}

Response (Error): 

{
  "message": "User already exists"
}

POST /api/login

Log in an existing user. Use this endpoint to authenticate a user and get a JSON Web Token (JWT) for further requests.

Request Body: 

{
  "email": "johndoe@example.com",
  "password": "password123"
}

Response (Success): 

{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}

Response (Error): 

{
  "message": "Invalid email or password"
}

GET /api/me

Get user information for the authenticated user. Use this endpoint after obtaining a JWT from the login process to access user-specific data.

Request Header: 

Authorization: Bearer <JWT Token>

Response (Success): 

{
  "id": 1,
  "name": "John Doe",
  "email": "johndoe@example.com"
}

Response (Error): 

{
  "message": "Authorization denied"
}
